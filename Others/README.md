# 📁 Others — Espace de contributions libres

Bienvenue ! Ce dossier est fait pour que **tout le monde puisse contribuer**, même sans connaître Git.

---

## ✅ Comment déposer un fichier ici ?

> Tu dois avoir été ajouté(e) au dépôt par Alexandre au préalable.

**Depuis cette page GitHub :**

1. Clique sur **"Add file"** (en haut à droite de la liste de fichiers)
2. Choisis **"Upload files"**
3. Glisse-dépose ton fichier ou clique pour le sélectionner
4. En bas de la page, laisse le message par défaut ou écris quelque chose comme `Ajout fiche Maths - [ton prénom]`
5. Clique sur **"Commit changes"** ✅

C'est tout ! Ton fichier est en ligne.

---

## 📝 Comment nommer ton fichier ?

Pour que ce soit lisible pour tout le monde, essaie de suivre ce format :

```
[Matière]_[Sujet]_[TonPrénom].[extension]
```

Exemples :
- `Maths_Fonctions_Sophie.pdf`
- `Lettres_Argumentation_Karim.docx`
- `SVT_Cellule_Léa.pdf`
